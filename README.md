# Overview

ATHLETE FIT is a modern cross-platform fitness analytics application developed using Flutter and Dart to provide users with real-time health tracking, activity monitoring, and interactive fitness insights across Android and iOS devices.

The application integrates Firebase authentication, cloud synchronization, Riverpod state management, and responsive Material Design 3 UI components to deliver a scalable and smooth user experience.

Designed with clean architecture principles, the application separates presentation, domain, and data layers for maintainable and production-ready mobile application development.

# How It Works

1. User logs into the application using Firebase Authentication  
2. Activity and fitness data are tracked in real time  
3. Riverpod manages application state and UI updates  
4. Fitness analytics are processed and visualized using FL Chart  
5. Cloud synchronization stores and retrieves user health data  
6. Responsive dashboards display real-time fitness insights

# Installation

## Clone the repository

```bash
git clone https://github.com/Anagh-007/ATHLETE-FIT.git
cd ATHLETE-FIT
```

## Install dependencies

```bash
flutter pub get
```

## Run the application

```bash
flutter run
```
